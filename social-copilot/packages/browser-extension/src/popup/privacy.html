隐私政策（简版）

Social Copilot 会在网页聊天界面中运行，并根据你的设置生成回复建议。 生成建议时，扩展可能会将“必要的对话上下文”发送到你选择的第三方模型服务（例如 DeepSeek/OpenAI/Claude）。

本地存储

  • 配置与偏好默认存储在本地（浏览器扩展存储 / IndexedDB）。
  • API Key 默认不持久化；如你勾选“记住 API Key”则会本地保存。
  • 如启用“长期记忆”，扩展会本地存储联系人摘要（不展示原文消息）。

发送范围控制

  • 可在设置中开启“脱敏/匿名化”，并限制上下文条数与字符预算。
  • 未勾选“隐私告知确认”前，扩展不会调用第三方模型接口。

清除数据

  • 你可以在“关于”页点击“清除所有数据”删除本地记录并重置扩展状态。

提示：如需导出排障信息，可在“关于”页下载诊断 JSON。
