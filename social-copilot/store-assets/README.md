# Chrome Web Store 物料目录

> 本目录用于存放上架所需物料（截图/宣传图/文案拷贝），方便协作与版本管理。

建议结构：

- `screenshots/`：商店截图（PNG/JPG）
- `promotional/`：宣传图（如需要）
- `copy/`：商店文案快照（可直接拷贝提交）

## 物料建议（通用）

- 截图至少 4 张（建议 6 张），覆盖：
  - 设置页（含隐私与发送范围开关）
  - 生成面板（候选回复 + 思路卡片）
  - 联系人页（偏好统计 + 长期记忆）
  - 关于页（清除数据 + 诊断导出）
- 描述文案参考：`social-copilot/docs/STORE_COPY.md`
- 隐私政策 URL：部署后填写 `docs/privacy.html` 或 `docs/privacy.zh-CN.html`
