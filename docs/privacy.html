Privacy Policy (Social Copilot)

Home · 中文

Last updated: 2025-12-19

1. Who we are

Social Copilot is a browser extension that provides reply suggestions for chat websites (Telegram Web / WhatsApp Web / Slack Web).

2. Data we process

2.1 Local storage (in your browser)

By default, we store the following data locally (IndexedDB / chrome.storage):

  • Chat messages (for context and style learning)
  • Contact profiles (e.g., interests, communication preferences)
  • Style preferences (your selections)
  • Optional long-term memory summaries (off by default)
  • Extension settings (provider/model/language/privacy toggles)

2.2 Data sent to third-party model providers

When you trigger generation (or enable optional automation), the extension sends the necessary conversation context to the third-party model provider you configure (e.g., DeepSeek / OpenAI / Anthropic Claude).

Default outbound minimization and protections (configurable):

  • Only the latest N messages are sent (default 10; the current message is always included)
  • Per-message and total context size limits (default 500 / 4000 chars)
  • Redaction of common PII (email / phone / URL) by default
  • Sender name anonymization ("Me" / "Other") by default
  • The raw field is not sent

3. How we use data

  • Generate reply suggestions
  • Update local profiles and preferences
  • (Optional) Maintain local memory summaries

4. What we do not do

  • We do not upload your API key to any self-hosted server (there is no default backend)
  • We do not collect behavioral analytics by default

5. Third-party processors

Third-party model providers process the context you send according to their own terms and privacy policies. You should review those policies before use.

6. Your controls

You can configure provider/model, persistence of API key, language, long-term memory, and outbound privacy settings (redaction/anonymization/context limits) in the extension popup.

You can clear local data at any time using the "Clear all data" button.
