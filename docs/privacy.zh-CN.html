隐私政策（Social Copilot）

首页 · English

最后更新：2025-12-20

1. 产品说明

Social Copilot 是一款为 Telegram Web / WhatsApp Web / Slack Web 等聊天网站提供“回复建议”的浏览器扩展。

本项目默认不包含任何自建后端；你需要自行配置第三方模型服务（如 DeepSeek / OpenAI / Anthropic Claude）的 API Key。

2. 我们处理哪些数据

2.1 浏览器本地存储

扩展默认在浏览器本地（IndexedDB / chrome.storage）存储：

  • 聊天消息（用于上下文与风格学习）
  • 联系人画像（兴趣/沟通偏好等）
  • 风格偏好（你的选择历史）
  • 扩展设置项（提供商/模型/语言/隐私开关等）
  • （可选）长期记忆摘要（默认关闭，需要你开启）

2.2 发送到第三方模型服务

当你触发生成（或开启相关功能）时，扩展会将“必要的对话上下文”发送到你选择的第三方模型服务，用于生成候选回复或提取结构化信息。

默认启用的发送侧保护（可在设置中调整）：

  • 仅发送最近 N 条消息（默认 10；当前消息始终包含）
  • 限制单条消息与总上下文字符预算（默认 500 / 4000）
  • 默认脱敏常见敏感信息（邮箱/手机号/链接）
  • 默认匿名化昵称（用“我/对方”替代）
  • 不发送消息的 raw 字段

3. 我们如何使用数据

  • 生成回复建议
  • 在本地更新联系人画像与风格偏好
  • （可选）在本地维护长期记忆摘要

4. 我们不会做什么

  • 不会把你的 API Key 上传到任何自建服务器（默认无后端）
  • 默认不做用户行为埋点/分析

5. 第三方处理者

第三方模型服务会按其服务条款与隐私政策处理你发送的上下文。使用前请先阅读对应提供商政策。

6. 你的控制权

你可以在扩展设置中配置：模型提供商/模型名称、是否持久化 API Key、语言、长期记忆开关，以及脱敏/匿名化与发送上限。

你可以随时在“关于”页点击“清除所有数据”删除本地存储内容。
